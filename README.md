Mi Clima App
Este es un proyecto para la materia de Desarrollo Móvil.

Es una app simple hecha en React Native (Expo) que te deja crear una cuenta, buscar el clima de cualquier ciudad y guardar tus ciudades favoritas.

Tecnologías Usadas
React Native (Expo): Para construir la app.

Firebase Authentication: Para el login y registro de usuarios (es la parte de seguridad).

Cloud Firestore: Es la base de datos en la nube donde se guarda la lista de favoritos.

OpenWeatherMap API: Es la API de terceros que usé para jalar la información del clima en tiempo real.

Para Correr el Proyecto
Instalar todo con npm install.

Poner tus claves en el archivo firebaseConfig.js (para Firebase) y en app/(tabs)/index.js (para la API del clima).

Correr con npm start.

Escanear el QR con la app de Expo Go.
